Конечно! Вот общее описание программы, реализующей сайт "Карта добрых дел", на основе предоставленного Технического задания и разработанного кода:

Общее описание программы: "Карта добрых дел"
"Карта добрых дел" — это одностраничное веб-приложение (SPA), разработанное с использованием Node.js (бэкенд) и HTML/CSS/JavaScript (фронтенд), с локальной базой данных SQLite. Основная цель приложения — объединить информацию о НКО и волонтёрских организациях, особенно в малых городах присутствия Госкорпорации "Росатом", чтобы повысить их видимость и вовлечённость жителей в добровольческую деятельность.

Функциональные возможности:
Просмотр НКО без регистрации:
Пользователи могут заходить на сайт и сразу видеть интерактивную карту.
На карте отображаются метки как для городов присутствия Росатом, так и для НКО, данные которых уже одобрены администратором.
Доступна навигация по карте (масштабирование, панорамирование).
Фильтрация и поиск:
Пользователи могут фильтровать НКО по городу и категории деятельности (например, благотворительность, экология).
Доступен поиск по названию и описанию НКО.
Регистрация и авторизация пользователей:
Пользователи могут создать аккаунт, указав email, имя и фамилию.
Для входа используется аутентификация по логину и паролю с генерацией JWT-токена для последующих защищённых запросов.
Реализована функция восстановления пароля через email.
Добавление и редактирование НКО:
Только авторизованные пользователи могут предложить добавление новой НКО или отредактировать уже существующую (при условии, что они её создали).
Форма добавления/редактирования включает поля: название, категория, описание, контакты, адрес, сайт, соцсети (в формате JSON), координаты, файл логотипа.
Все предложения проходят предварительную модерацию: они сохраняются в очередь (moderation_queue) и не отображаются на карте до тех пор, пока администратор не одобрит их.
Просмотр карточки НКО:
При клике на метку НКО на карте открывается информационная панель, где отображаются все сохранённые данные: название, категория, описание, контакты, ссылки на сайт и соцсети, логотип.
Технические особенности:
Локальная база данных SQLite используется для хранения информации о пользователях, НКО, заявках на модерацию и других данных.
Бэкенд на Node.js обрабатывает API-запросы от фронтенда, взаимодействует с БД, обеспечивает аутентификацию и логику модерации.
Фронтенд реализован как одностраничное приложение, обеспечивающее плавное взаимодействие без перезагрузки страницы.
Интерактивная карта реализована с использованием Яндекс.Карт API.
Интерфейс адаптивен и корректно отображается на устройствах с различными размерами экрана.
Код структурирован (многоуровневая архитектура) для дальнейшей доработки и масштабирования.
Цель проекта:
Создать удобный, интуитивно понятный и визуально привлекательный инструмент, который поможет жителям, волонтёрам и представителям бизнеса найти, связаться и взаимодействовать с НКО в их регионе, тем самым способствуя росту социальной активности и поддержке добрых дел.
